Repository Name: Simon-Game

Description:

Welcome to the my Simon Game Repository!

Overview:
My Simon Game is a classic memory and pattern 
recognition game inspired by the popular Simon game. 
Test and improve your memory skills by repeating a 
sequence of colors that the game generates. 
The challenge escalates with each successful round, 
putting your memory and concentration to the test.

Features:

Intuitive and user-friendly gameplay.
Engaging and addictive pattern repetition challenges.
Colorful and visually appealing design.
Sound effects for an immersive gaming experience.
Adjustable difficulty levels for players of all ages.
Score tracking and level progression

Technologies Used:

HTML5 for the structure and layout.
CSS3 for styling and animations.
JavaScript for game logic and interactivity.
